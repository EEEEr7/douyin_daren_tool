一、工具简介
================================================================================

本工具从抖音「精选联盟 · 百应后台」（buyin.jinritemai.com）的「达人广场」
中，自动采集符合条件的达人微信号，并导出 Excel。

【默认筛选】
  · 主推类目：3C数码家电
  · 达人信息：有联系方式

【单次目标】
  · 每次凑满 20 条微信号后自动停止
  · Excel 仅含成功项：达人昵称、微信号
  · 已采集 UID 自动去重


================================================================================
二、环境要求
================================================================================

  操作系统    Windows 10 / 11（64 位）
  网络        可访问 buyin.jinritemai.com
  账号        百应/精选联盟权限（不是抖店 fxg）
  浏览器      Microsoft Edge（全程使用）

  建议：Edge 最大化，显示缩放 100%～125%


================================================================================
三、安装与启动
================================================================================

【发布包解压后主要文件】
  XTeink 抖音达人微信采集.exe
  启动.bat / 启动Edge.bat
  使用说明.txt / 使用说明书.md / 使用说明书.txt
  data/  output/

【启动程序】
  双击「启动.bat」或 exe


================================================================================
四、连接 Edge 与登录（无需在程序内再登录）
================================================================================

【推荐流程】

  1. 双击「启动Edge.bat」（调试端口 9222，打开达人广场）
  2. 在 Edge 登录 buyin.jinritemai.com（百应，只需一次）
  3. 双击「启动.bat」打开采集程序
  4. 点「连接 Edge」（可能短暂重启 Edge，登录保留）
  5. 显示「Edge 已连接」后，点「开始采集」

【按钮说明】
  连接 Edge      建立连接
  检查连接       验证 Edge + 百应登录
  重新验证登录   勾选后采集前再检查

【注意】
  · 从任务栏打开的 Edge 可能连不上，请用「启动Edge.bat」
  · 登录百应 buyin.jinritemai.com，不是抖店


================================================================================
五、开始采集
================================================================================

  1. 确认 Edge 已连接
  2. （可选）选择运行时段
  3. 点「开始采集」
  4. 看「本次进度」与运行日志
  5. 完成后点「打开结果文件夹」

【自动步骤】
  验证登录 → 筛选 3C+有联系方式 → 进达人主页
  → 左侧「其他信息」栏点小眼睛 → 凑满 20 条 → 导出 Excel

【采集中请勿】关闭程序、手动切换 Edge 标签


================================================================================
六、结果文件

  目录：output/XTeink_YYYYMMDD_HHMM_时段/
  文件：XTeink_达人联系方式.xlsx

  列：达人昵称 | 微信号（仅成功项）


================================================================================
七、常见问题
================================================================================

【无法连接 Edge】
  关闭 Edge → 双击「启动Edge.bat」→ 程序内「连接 Edge」

【登录失败】
  Edge 重新登录 buyin.jinritemai.com →「检查连接」

【采集 0 条】
  · 达人均已采集过（删 processed_uids.json 可重采）
  · 看日志 [跳过] 原因

【小眼睛/微信号异常】
  程序会校验格式，无效值（如「首页」）不写入 Excel
  持续失败请联系 Er7 并提供日志

【不同分辨率】
  可用，建议 Edge 最大化
