## 一、工具简介

本工具用于从抖音 **精选联盟 · 百应后台**（[buyin.jinritemai.com](https://buyin.jinritemai.com)）的 **达人广场** 中，自动采集符合条件的达人 **微信号**，并导出为 Excel 表格。

**默认筛选条件：**

| 项目 | 设置 |
|------|------|
| 主推类目 | 3C数码家电 |
| 达人信息 | 有联系方式 |

**单次采集目标：**

- 每次运行 **凑满 20 条微信号** 后自动停止
- 仅导出 **成功读取** 的达人（失败项不写入 Excel）
- 已采集过的达人 **UID 自动去重**，不会重复抓取

## 二、环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 / 11（64 位） |
| 网络 | 可正常访问 buyin.jinritemai.com |
| 账号 | 已开通 **精选联盟 / 百应** 权限的抖音电商账号 |
| 浏览器 | **Microsoft Edge**（全程使用，复用登录态） |

> **重要：** 请登录 **百应**（buyin.jinritemai.com），不是抖店（fxg.jinritemai.com），两者不是同一系统。

**屏幕建议：**

- Edge 窗口 **最大化** 或宽度 ≥ 1280px
- 显示缩放 100%～125% 较稳定

## 三、安装与首次启动

### 3.1 解压发布包（同事版）

1. 解压 `XTeink_达人微信采集_v1.2.0.zip` 到任意目录（建议 `D:\XTeink`）
2. 解压后主要文件：

```
XTeink_达人微信采集/
├── XTeink 抖音达人微信采集.exe   ← 主程序
├── 启动.bat                      ← 双击启动程序
├── 启动Edge.bat                  ← 带调试端口启动 Edge（推荐）
├── 创建桌面快捷方式.bat
├── 使用说明.txt
├── 使用说明书.md / 使用说明书.txt
├── data/                         ← 采集记录（勿分享）
└── output/                       ← Excel 输出目录
```

### 3.2 开发/源码运行

```bash
pip install -r requirements.txt
playwright install chromium   # 仅非 Edge 备用路径需要
python main.py                # 或双击 start.bat
```

## 四、连接 Edge 与登录

程序通过 **连接本机 Edge** 复用登录态，**无需在程序内再次登录**。

### 推荐流程

1. 双击 **`启动Edge.bat`**（会以调试端口 9222 打开 Edge，并打开达人广场）
2. 在 Edge 中登录 **buyin.jinritemai.com**（若尚未登录）
3. 双击 **`启动.bat`** 打开采集程序
4. 点击 **「连接 Edge」**（程序可能短暂重启 Edge，**登录状态会保留**）
5. 状态显示 **「Edge 已连接 · 可直接采集」** 后，点击 **「开始采集」**

### 界面按钮说明

| 按钮 | 作用 |
|------|------|
| 连接 Edge | 建立与 Edge 的连接（必要时短暂重启 Edge） |
| 检查连接 | 验证 Edge 是否已连接且百应已登录 |
| 开始前重新验证登录 | 勾选后，采集前再次检查登录态 |

### 常见问题

- **从任务栏直接打开的 Edge，程序可能连不上** → 请用 `启动Edge.bat` 或程序内「连接 Edge」
- **提示无法连接** → 关闭所有 Edge 窗口后，重新双击 `启动Edge.bat`

## 五、开始采集

### 5.1 图形界面操作

1. 确认 **Edge 已连接**
2. （可选）设置 **运行时段**：自动 / 上午 / 下午 / 晚上
3. 点击 **「开始采集」**
4. 观察 **运行日志** 与 **本次进度** 进度条
5. 完成后点击 **「打开结果文件夹」** 查看 Excel

### 5.2 采集过程说明

程序会自动完成：

1. 验证百应登录态
2. 打开达人广场，筛选 **3C数码家电 + 有联系方式**
3. 读取符合「有联系方式」的达人列表
4. 跳过 `data/processed_uids.json` 中已采集 UID
5. 逐个进入达人主页，在左侧 **「其他信息」** 栏点击 **微信号旁小眼睛** 读取明文
6. 凑满 **20 条** 成功微信号后停止，导出 Excel

**采集过程中：**

- 请勿关闭程序窗口
- 尽量不要手动切换 Edge 标签页或干扰自动化页面

### 5.3 运行日志示例

```

已点击筛选：3C数码家电 + 有联系方式
```

1. **勿分享 `data/` 文件夹** — 含采集记录与本地数据

4. **保持网络畅通** — 采集需稳定访问百应后台

## 八、常见问题

### Q1：程序打不开 / 闪退

- 右键 exe → **以管理员身份运行**
- 杀毒软件添加信任后重试

### Q2：无法连接 Edge

- 关闭所有 Edge 进程后，双击 **`启动Edge.bat`**
- 在程序中点击 **「连接 Edge」**

### Q3：登录验证失败

- 在 Edge 打开 buyin.jinritemai.com 重新登录百应
- 点击 **「检查连接」** 确认状态

### Q4：采集结果为空 / 提示未获取到微信号

- 列表中达人均已采集过（UID 去重）→ 可删除 `data/processed_uids.json` 后重试（会重复采集）
- 当前页符合筛选的达人已用尽 → 稍后重试或换页
- 查看运行日志中 `[跳过]` 原因

### Q5：小眼睛点了但微信号不对

- 程序已校验微信号格式，**「首页」等无效文字不会写入 Excel**
- 若持续失败，请提供运行日志与屏幕分辨率信息联系技术支持

### Q6：不同分辨率电脑能用吗？

- 可以。程序按页面文字和 DOM 定位，**不依赖固定坐标**
- 建议 Edge 最大化，缩放 100%～125%
